# partstoob

Experimental branch porting archinstoo to run from **any Linux host** (not just Arch ISOs).

| Host | `arch-install-scripts` | `pacman` | Script |
|------|------------------------|----------|--------|
| Alpine | ✅ | ✅ | [ALP](./ALP) |
| Debian | ✅ | ✅ | [DEB](./DEB) |
| Fedora | ✅ | ✅ | [FED](./FED) |
| NixOS | ✅ | ✅ | - |

See [BOOTSTRAP.md](.github/BOOTSTRAP.md) for details.
